# TapSwap Automation System

Высоконагруженный асинхронный Python‑софt для работы с большим количеством аккаунтов в Telegram‑приложении TapSwap.  
Проект был рассчитан на стабильную массовую работу с прокси, рандомизацией действий, имитацией пользовательского поведения и минимизацией блокировок.

## Ключевые особенности

- асинхронная обработка большого числа аккаунтов;
- работа через прокси и распределение нагрузки;
- рандомизация действий и сценариев поведения;
- имитация естественной активности пользователей;
- поддержка нескольких типов аккаунтов и разных рабочих сценариев;
- ориентирован на устойчивую работу в условиях ограничений и антибот‑механизмов.
